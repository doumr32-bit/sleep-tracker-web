# Sleep Tracker Web

Sleep tracking web app built with FTWNOW full workflow and MCP-gated delivery.

一个轻量睡眠记录 Web 应用，按 FTWNOW 全流程交付。

## 本地运行

```bash
npm test
npm start
# 打开 http://127.0.0.1:4173
```

## 功能

- 记录睡眠（日期、入睡、醒来、质量、备注）
- 自动计算睡眠时长（支持跨午夜）
- 统计聚合（平均时长、平均质量、最佳/最短）
- 本地持久化（localStorage）
